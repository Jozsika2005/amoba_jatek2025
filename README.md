# Amőba játék – Java beadandó

## Leírás
Ez a projekt egy konzolos amőba (tic-tac-toe) játék Java nyelven megvalósítva.  
A program objektumorientált elvek szerint készült, Maven alapú build rendszerrel.

A játék támogatja a változtatható táblaméretet, kezeli az érvénytelen lépéseket,
és megállapítja a győzelmet vagy a döntetlent.

---

## Követelmények teljesítése

- Java (objektumorientált megvalósítás)
- Maven projekt
- JUnit 5 tesztek
- JaCoCo kódfedettség (80% felett)
- Checkstyle kódkonvenciók
- Saját kivétel használata
- Konzolos felhasználói felület
- Futtatható alkalmazás `main` metódussal
